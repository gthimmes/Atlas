# Keyboard shortcuts

## Navigation

| Key | Action                                    |
| --- | ----------------------------------------- |
| `G` | Jump to Work Graph                        |
| `S` | Jump to Spec Editor (for the active spec) |
| `R` | Jump to Agent Run Panel _(Phase 3)_       |
| `D` | Jump to Morning Digest _(Phase 4)_        |

Single-letter shortcuts are suppressed while typing in an `<input>` or
`<textarea>`.

## Help

| Key     | Action                                      |
| ------- | ------------------------------------------- |
| `?`     | Open / close this help drawer               |
| `Esc`   | Close the help drawer or modals             |
| `1`–`8` | Jump to help section (while drawer is open) |
